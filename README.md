## Database Setup

1. **Create the database:**
    ```bash
    php artisan db:create
    ```

2. **Migrate the database tables:**
    ```bash
    php artisan migrate
    ```

3. **Seed the database:**
    ```bash
    php artisan db:seed
    ```
